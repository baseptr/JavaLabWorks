# Lab: Java HTTP Client

## Objective

Разработать консольное приложение, которое отправляет HTTP-запросы к внешнему API с помощью встроенного в Java
HTTP-клиента (`java.net.http.HttpClient`), считывая команды из консоли в бесконечном цикле.

## Tasks

1. Реализовать консольный цикл, считывающий команды из консоли и выполняю[щий соответствующие HTTP-запросы
2. Завершать выполнение программы при вводе команды `exit`](1. HTTP client
    - Написать приложение, отправляющее в цикле запросы на URL при промощи встроеного в Java HTTP клиента
    - Выполнять отправку в бесконечном цикле, считывая команды из консоли
    - Завершать выполнение по команде Exit
    - Выводить полученые данные в консоль
    - Get 1, Get ALL, Delete, Post, Put
    - https://jsonplaceholder.typicode.com/todos/
    - SOLID, KISS

   Дополнительное задание:

    - Использовать любое другое API
    - отправлять запросы к нескольким ресурсам по вводу из консоли
    - Использовать Jackson или Gson для парсинга JSON-ответов и настроить ингорирование одного поля и кастомный формат
      для еще одного поля желательно даты
3. Реализовать следующие команды:
    - `get-all` — получить список всех ресурсов (`GET /todos`)
    - `get` — получить один ресурс по id (`GET /todos/{id}`)
    - `post` — создать новый ресурс (`POST /todos`)
    - `put` — обновить существующий ресурс (`PUT /todos/{id}`)
    - `delete` — удалить ресурс (`DELETE /todos/{id}`)
4. Выводить полученный ответ (статус-код и тело) в консоль
5. Использовать `https://jsonplaceholder.typicode.com/todos/` в качестве базового URL
6. Соблюдать принципы **SOLID** и **KISS** при проектировании приложения

## Deliverables

- Класс для работы с HTTP-клиентом (`HttpClient`)
- Отдельный класс или интерфейс для команд
- Консольный цикл считывания команд
- Вывод результатов запросов в консоль

## Acceptance Criteria

- Программа запускается и ожидает ввода команды
- Каждая команда (`get-all`, `get`, `post`, `put`, `delete`) корректно выполняет HTTP-запрос
- Ответ сервера (статус + тело) выводится в консоль
- Ввод `exit` завершает программу
- Код структурирован в соответствии с принципами SOLID и KISS

## Additional Tasks

- Использовать другое публичное API (например `https://jsonplaceholder.typicode.com/posts/` или любое другое)
- Предоставить возможность выбора ресурса из консоли (например: `todos`, `posts`, `users`) и отправлять запросы к
  выбранному ресурсу
